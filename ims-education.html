IMS Courses

Build your skills with these self-paced, no-cost courses and earn a digital badge.

IMS Fundamentals

Learn about the main components of IBM Information Management System (IMS): IMS Database Manager (IMS DB), IMS Transaction Manager (IMS TM), and a set of system services that provide common services to IMS DB and IMS TM. You'll how these components work and interact with application programs, how IMS databases differ from each other and differ from relational databases, how to connect to non-IMS systems, how and when to use logging, restarts, security features, and more. If you successfully complete the course, you'll earn a digital badge!

This self-paced, no cost course is for users who are new to IMS and want to get basic but comprehensive training on IMS architecture, services, hierarchical databases, transaction processing, application development, and more.

Learn

IMS Diagnostics

Learn about the relationships between IMS components so that you can explain IBM's recommendations for IMS diagnostic setup. You'll learn how to use the IMS Interactive Dump Formatter, identify the key z/OS IPCS commands that are useful for IMS problem diagnostics, perform the methodology for debugging IMS abends, and understand the methodology for debugging IMS waits and loops. If you successfully complete the course, you'll earn a digital badge!

Learn

IMS Systems Programming in a DBCTL Environment

Learn about the IMS database control (DBCTL) environment, which provides access between CICS and IMS-managed databases. You'll learn how to install, configure, maintain, troubleshoot, and support an IMS DBCTL system, and understand the IMS architecture that applies to database management and attaching to CICS systems and transactions. If you successfully complete the course, you'll earn a digital badge!

Learn

IMS Shared Queues

In this no-cost, self-paced course, you'll learn how shared message queues work in IBM IBM Information Management System (IMS), and how full-function queues and Fast Path expedited message handler (EMH) queues are shared within a parallel sysplex environment. You'll learn how to prepare and implement IMS shared queues and manage and troubleshoot an IMS shared queues environment.

Have any questions about IMS courses, or the badge program? Here are some quick answers that may help you.

How do I earn a badge?

Courses that offer badges include one or more exams. You must pass those exams. See the "Grading and badge" topic in the course to see the grading policy and how exams are graded.

If I pass the course, when do I get the badge?

Badges are typically issued automatically after you meet the course requirements and grading policy. You should get an email notification of receiving the badge in one to four days.

Whom do I contact if I have problems with course content?

You can contact the IMS Skills Development team at ibmimsom@us.ibm.com.

Where can I find more information about digital badges?

For more information about badges, see Frequently asked questions on IBM Training.
